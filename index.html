Case Study · Lumoura

Le problème

Créer une plateforme e-commerce de parfums de luxe complète et fonctionnelle, avec gestion des utilisateurs, des commandes et une interface d'administration robuste — le tout sans framework back-end, en PHP pur avec architecture MVC.

La solution

Architecture MVC maison avec routeur personnalisé, authentification sécurisée par OTP email, API REST pour l'autocomplétion d'adresse (API Adresse gouv.fr), génération de factures PDF avec mPDF, et un dashboard admin complet pour gérer produits et commandes.

Technologies utilisées

PHP 8MySQLArchitecture MVC API RESTOTP EmailmPDF PDO sécuriséJavaScriptCSS3

Difficultés rencontrées

  • Implémenter le système OTP avec expiration et renouvellement sécurisé
  • Gérer les sessions et la sécurité des requêtes SQL avec PDO
  • Intégrer mPDF pour générer des factures PDF dynamiques formatées
  • Concevoir un routeur MVC sans framework depuis zéro

Résultat final

Plateforme e-commerce 100% fonctionnelle avec catalogue produits, panier, système de commandes complet, espace client, génération de factures PDF téléchargeables et interface admin pour gérer l'ensemble du site. Projet phare du BTS SIO SLAM.
